About Project

The project focuses on augmenting the functionality of a Python application that encompasses AWS EC2 instance management and gesture recognition using computer vision techniques. While the Python application itself provides robust capabilities, its utilization is often limited to users with programming expertise. By developing a web interface powered by the Flask web framework, this project aims to democratize access to these functionalities, enabling users to interact with the system through a familiar and intuitive browser-based interface.

Through the web interface, users will be able to perform essential tasks such as launching and terminating EC2 instances on AWS, as well as initiating gesture recognition processes. Leveraging the power of Flask, coupled with HTML templates and CSS styling, the interface will offer a seamless user experience, allowing users to execute complex operations with minimal effort and technical expertise.

Technologies Used

- Programming language: Python
- Web framework: Flask
- Libraries: boto3, cvzone, OpenCV
